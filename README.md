# Ping-Pong-LED-Uygulamas-
Bu projede PIC16F628A mikrodenetleyicisi kullanılarak LED’lerin sağa ve sola kaydırılması gerçekleştirilmiştir. Assembly dili ile yazılan programda, bit kaydırma komutları kullanılarak LED’ler üzerinde hareketli bir ışık efekti oluşturulmuştur. Yön kontrolü bir değişken ile sağlanmış, gecikme alt programı ile hareket hızı ayarlanmıştır.
